Mi Menu

Mi Menu es una página web de comidas donde se podrá ver el menú de diferentes restaurantes. Los usuarios logueados dependiendo de su rol tendran diferentes funcionalidades: Restaurantes: podran crear comidas (editarlas y eliminarlas). Comensales: podran ver los menus de los restaurantes y valorarlos.

Esta página web fue creada como pryecto final del curso de "Programación Web Full-Stack" 2023 de SILICON Misiones.
